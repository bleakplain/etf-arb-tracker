# A股涨停ETF溢价套利策略说明

## 策略概述

当个股涨停无法买入时，通过买入持有该个股的ETF来获取溢价收益。

### 核心原理

1. **个股涨停**：个股涨停后无法直接买入
2. **ETF替代**：找到持仓中包含该涨停股的ETF
3. **溢价传递**：个股涨停带动ETF净值上涨
4. **收益获取**：通过买入ETF获得该个股涨停的收益

---

## 策略条件

### 必要条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 个股状态 | 涨停 | 涨跌幅达到涨停限制 |
| 持仓占比 | ≥ 5% | 该股票在ETF中的持仓占比必须达到5%以上 |
| 持仓排名 | 前10名 | 股票必须在ETF前十大持仓中 |

### 辅助条件

| 条件 | 要求 | 说明 |
|------|------|------|
| 时间限制 | 距收盘 ≥ 30分钟 | 避免尾盘涨停的不确定性 |
| ETF流动性 | 日成交额 ≥ 5000万 | 确保能够顺利买入卖出 |
| 封单量 | ≥ 10亿 | 封单量越大，涨停越可靠 |

---

## 信号评估

### 置信度评估

| 置信度 | 条件 |
|--------|------|
| 高 | 权重 ≥ 10% 或 排名 ≤ 3 |
| 中 | 5% ≤ 权重 < 10% |
| 低 | 权重 < 5% |

### 风险等级评估

| 风险等级 | 条件 |
|----------|------|
| 低 | 距收盘 > 1小时 |
| 中 | 距收盘 10分钟-1小时 |
| 高 | 距收盘 < 10分钟 或 前10持仓占比 > 70% |

## 风险提示

### 主要风险

1. **涨停打开风险**
   - 个股涨停后可能打开，导致ETF跟随下跌
   - 缓解：选择封单量大的涨停股

2. **溢价率风险**
   - ETF本身存在溢价，可能损失溢价收益
   - 缓解：选择溢价率低的ETF

3. **尾盘风险**
   - 尾盘涨停不确定性大，第二天可能低开
   - 缓解：设置距收盘30分钟的时间限制

4. **持仓集中度风险**
   - ETF持仓过于集中，受单一股票影响大
   - 缓解：关注前10持仓占比，选择分散度高的ETF

5. **流动性风险**
   - ETF流动性不足，可能无法顺利买卖
   - 缓解：设置日成交额下限

## 策略参数配置

```yaml
# config/settings.yaml

strategy:
  # 最小持仓权重（5%）
  min_weight: 0.05

  # 最小封单量（10亿元）
  min_order_amount: 10

  # 距收盘最小时间（30分钟=1800秒）
  min_time_to_close: 1800

  # ETF最小日成交额（5000万元）
  min_etf_volume: 5000

risk_control:
  # 单次最大买入金额（50万元）
  max_buy_amount: 50

  # 止盈点（3%）
  take_profit: 3

  # 止损点（-2%）
  stop_loss: -2

  # 持仓天数上限（3天）
  max_hold_days: 3
```

## 交易执行

### 买入时机

1. 个股涨停确认
2. ETF持仓占比 ≥ 5% 验证通过
3. 距收盘时间 ≥ 30分钟
4. ETF流动性充足

### 卖出时机

1. **止盈**：ETF涨幅达到 3%
2. **止损**：ETF跌幅达到 -2%
3. **时间止损**：持仓超过 3 个交易日
4. **个股打开**：涨停股打开且跌幅 > -5%

## 总结

该策略适用于：
- 短线套利交易
- 无法直接买入涨停股的场景
- 希望获取涨停股收益的投资者

策略优势：
- 利用ETF替代直接买入个股
- 分散个股风险
- 流动性好，易于进出

注意事项：
- 严格遵循持仓占比 ≥ 5% 的要求
- 注意时间限制，避免尾盘风险
- 设置止盈止损，控制风险
