## ETF基础知识

### ETF类型
1. **宽基指数ETF**: 沪深300, 中证500, 上证50等
2. **行业ETF**: 半导体, 新能源, 医药等
3. **策略ETF**: Smart Beta, 分红等
4. **跨境ETF**: 恒生, 纳指等

### 关键指标
- **IOPV (Indicative Optimized Portfolio Value)**: 实时参考净值
- **溢价率**: (市价 - IOPV) / IOPV
- **折价率**: (IOPV - 市价) / IOPV
- **跟踪误差**: ETF与基准指数的偏离度

### 流动性评估
- 日成交额 > 5000万: 流动性良好
- 日成交额 > 1亿: 流动性优秀
- 日成交额 < 1000万: 流动性差，避免

## 套利策略原理

### 基本原理
当成分股涨停无法买入时，通过购买ETF间接持有该股票。

### 套利效果计算
```
套利效果 = (股票涨停涨幅 - ETF涨幅) × 股票在ETF中的权重
```

### 例子
- 贵州茅台涨停 +10%
- 沪深300ETF中茅台权重 5%
- ETF涨幅假设 +2%
- 套利效果 = (10% - 2%) × 5% = 0.4%

### 风险因素
1. **溢价风险**: ETF本身溢价买入会亏损
2. **跟踪误差**: ETF可能不完全跟踪
3. **时间衰减**: 越接近收盘，套利效果越弱
4. **流动性风险**: ETF流动性不足无法卖出

## 信号评估标准

### 高质量信号
- ✅ 股票确认涨停
- ✅ ETF权重 >= 8%
- ✅ ETF日成交额 > 1亿
- ✅ 距离收盘 > 60分钟
- ✅ ETF溢价率 < 1%

### 中等质量信号
- ⚠️ 股票接近涨停 (+9%以上)
- ⚠️ ETF权重 5-8%
- ⚠️ ETF日成交额 5000万-1亿
- ⚠️ 距离收盘 30-60分钟

### 低质量信号
- ❌ ETF权重 < 5%
- ❌ ETF日成交额 < 5000万
- ❌ ETF溢价率 > 2%
- ❌ 距离收盘 < 30分钟

## 策略参数说明

### highest_weight (最高权重策略)
- 选择权重最高的ETF
- 优点: 套利效果最大
- 缺点: 可能流动性差

### balanced (平衡策略)
- 综合考虑权重、流动性、溢价率
- 评分 = 权重 × 流动性因子 / (1 + 溢价率)

### best_liquidity (最佳流动性策略)
- 优先选择流动性最好的ETF
- 优点: 容易进出
- 缺点: 套利效果可能较低

## 监管注意事项

### T+1交易制度
- 当日买入次日才能卖出
- 套利需要考虑次日开盘表现

### 涨停限制
- 连续涨停可能被停牌核查
- 注意监管公告

### ETF申购赎回
- 最小申购赎回单位通常为100万份
- 个人投资者通常用二级市场交易
